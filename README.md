# AI Stack Lab — Image Assets

Public image assets for the [AI Stack Lab](https://www.aistacklabhq.com) blog, served via the jsDelivr CDN.

## Usage

Covers live under `covers/`. Reference them in blog HTML via jsDelivr:

```
https://cdn.jsdelivr.net/gh/JGUY87/aistacklab-assets@main/covers/<file>.webp
```

- WebP only (speed / Core Web Vitals).
- `@main` resolves to the latest commit on the default branch (CDN-cached ~7 days). For an immutable URL, pin a commit SHA instead of `@main`.

This repo holds only public blog cover images — no source code, no secrets.
